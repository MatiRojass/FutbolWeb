Sobre el Proyecto

Este proyecto fue desarrollado como práctica para mejorar mis habilidades en el consumo de APIs con JavaScript y profundizar en el uso de tecnologías web modernas.

Características

  • Consumo de la API de Football API para obtener datos en tiempo real.
  • Actualización automática de marcadores cada 2.5 minutos.
  • Uso de localStorage para almacenar datos estáticos, como la tabla de posiciones de la liga argentina 2021.
  • Implementación de componentes reutilizables como el header con un menú desplegable.
  • Exploración y uso de plantillas HTML (<template>) para generar dinámicamente tarjetas de partidos.

Tecnologías Utilizadas

  • HTML5 y CSS3 para la estructura y el diseño visual.
  • JavaScript (Vanilla) para la lógica y las interacciones dinámicas.
  • Vite para optimizar y servir la aplicación durante el desarrollo.

Cómo Funciona

La aplicación tiene dos secciones principales:

  1. Partidos en Vivo: Muestra los marcadores de los partidos en tiempo real con actualizaciones automáticas cada 2.5 minutos.
  2. Tabla de Posiciones: Presenta la tabla estática de la liga argentina 2021, utilizando localStorage para optimizar el acceso a los datos.
